# Flutter Weather App ☁️

A modern, dark-mode Weather Application built with Flutter that provides real-time weather updates and forecasts.

## ✨ Features
* **Current Weather:** View live temperature and weather conditions (Rain, Clouds, etc.).
* **Forecast:** 24-hour horizontal forecast scroll showing temperature trends.
* **Detailed Metrics:** Track humidity, wind speed, and atmospheric pressure.
* **Responsive Design:** Optimized for a sleek mobile experience on iOS and Android.

## 🛠️ Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **API:** OpenWeatherMap (or similar)
